# mooc_crawler
An implementation of BIT-IR final project crawler part
中国大学MOOC 基于scrapy 的多线程爬虫

## Getting Started
```shell script
$ virtualenv venv
$ . venv/bin/activate
$ pip install -r requirements.txt
```

## Prerequisites
- Python >= 3.6
- Docker >= 19.03.3-ce
- docker-compose >= 1.24.1

## Run

```shell script
$ python app.py
```

## Deployment

```shell
$ make run
```
